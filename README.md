## HOW TO USE
1. Go to the Releases section in this repo to the right --->
2. Download the latest release of the Unity Package.
3. Drag-and-drop the Unity Package into your Unity Project.
4. Follow the tutorial: https://docs.google.com/document/d/1ew-wYyu2hF7ZvEidzL7bxC4gf9q_BOA7A6OS6rFNwPA/edit
5. Start building!

## UNITY TEMPLATE

This template is to facilitate developers in setting up a project where they can build a game that has ***Boom Dao Login*** integration ready and ***Boom Dao Gaming Guilds*** integration ready to quickly build minigames that can leverage the Boom Protocol ecosystem of thousands of daily users.
